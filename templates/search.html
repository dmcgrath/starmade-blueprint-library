What is Class Rank?

Class Rank is a classification system based on the calculated mass of the entity.

What is Material Simplicity?

This is a count of unique block types used in the blueprint. Currently it is only for the base entity (excludes docked entities). So searching for Blueprints with Material Simplicity <= 50 will return only Blueprints where the base entity uses 50 or fewer block types.
